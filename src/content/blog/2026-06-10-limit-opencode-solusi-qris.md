## Limit Token — Musuh Kaum 0 Nol Rupiah

Beberapa hari lalu, aktivitas vibe coding dan ngulik AI agent saya berhenti total. Lagi asik ngoding, lagi seru-serunya ngutak-atik komponen Astro, tiba-tiba... *freeze*.

OpenCode free tier udah kena batas limit nya.

Xiixixix — resiko klasik kaum 0 nol rupiah. Saya cuma bisa geleng-geleng kepala sambil ngopi di depan laptop.

Halo semuanya, Boim di sini. Ceritanya sederhana: saya lagi pacak-pacak coding pake OpenCode, sebuah *agentic coding tool* yang belakangan ini jadi andalan saya. Masalahnya, saya pake free tier — tanpa top up, tanpa credit card, tanpa apa-apa. Gratisan murni.

Nah, efek samping gratisan ya begini: kena limit. Pas lagi butuh banget, pas otak lagi *flow* dan tangan pengen ngetik terus, server ngasih kode error. Mode *read-only*. Gas ambyar.

Dari situ saya mulai mikir, "Gimana caranya lanjut tanpa drama beginian?"

### Skenario A: Pake Free Terus

Opsi pertama yang paling obvious: ya pake free tier lagi aja. Tunggu sampe limitnya reset, terus lanjut lagi.

![FreeLLMAPI](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEiEDAF92QfgVSKlxArFG2BAsvT1trqQV7K3aIjF6DDW9_pTUb25sF-V-9JRQXW4yViXBex_XEYJJRDO82IEzB7INVmzZPNHAqMtm7vgetNBPP3-tFGWH3ai8Ex3GT4KYNskiFmV5NNGwZnWg-a6HtErSH5EM6SotK4JRtQFL6TqDZMcOoekwccU2W7kUxhF/s800/Screenshot%202026-06-10%20160211.webp)

Sebenernya ini bisa, sih. Tinggal sabar nunggu. Tapi masalahnya, saya paham pola hidup saya sebagai *vibe coder* — kalau lagi *mood*, mau nulis kode berjam-jam dan bisa ngabisin ribuan token dalam sehari. Besoknya *mood* udah ilang, males banget.

Masalah hari ini — kena limit — bakal terulang lagi. Nggak ada jaminan.

### Skenario B: Credit Card Temen

Opsi berikutnya: berlangganan resmi. OpenCode itu top up minimum cuma 5 dollar. Murah, kan? 5 dollar doang buat dapet akses penuh ke agent AI yang kenceng.

Tapi... kendalanya di pembayaran. Harus pake credit card.

Nah, saya ini demografinya bapak-bapak yang masih belajar di sela-sela kesibukan. Kartu kredit? Belom punya. Atau udah punya tapi lagi males ngurus. *Uhuk.*

Sempet mau pake credit card punya temen. Temen yang website-nya kebetulan saya urusin — sriwijayagrafika.com, website advertising Palembang. Tapi saya tunda. Entah kenapa, ada rasa nggak enak. Pinjem credit card orang buat bayar langganan AI? Rasanya aneh.

Saya mutusin buat cari solusi lain dulu.

### Skenario C: Antigravity

Ngeliriklah ke Antigravity. Saya pikir, "Kayaknya ini cocok buat saya — pembayaranya gampang, banyak opsi, bahkan bisa bayar di mart-mart terdekat, atau pake QRIS."

Tapi setelah baca-baca lagi, saya inget: Antigravity ini AI flagship Google yang pake model besar. Model besar = token cepet abis kalo dipake ngoding berat. Gaya belajar saya yang suka eksperimen dan ngabisin banyak token sepertinya nggak cocok.

Bayangin aja: saya suka *trial and error*. "Coba generate ini, salah, ulang lagi, generate lagi." Dalam 10 menit, token bisa ludes puluhan ribu. Di Antigravity, itu artinya dompet menjerit.

Rasanya kurang pas. Skip.

### Skenario D: Ngelirik Shopee

Ini yang agak absurd sebenernya. Iseng-iseng buka Shopee, cari "API AI" — ternyata ada banyak yang jualan. Mulai dari token Gemini, Claude, sampe model-model open source.

Saya sempet tertarik. Tapi lagi-lagi ada rasa nggak percaya 100%. Gimana ya — kita bayar ke penjual, penjual yang kontrol platform-nya. Kalo tiba-tiba doi tutup toko? Token saya lenyap. Kalo tiba-tiba limitnya di-reset dari server pusat? Saya gabisa ngapa-ngapain. Resiko nya terlalu gede.

Tapi dari pencarian di Shopee ini, saya nemu sesuatu yang menarik.

### eureeka! iLabs — Toko yang Beda

![Shopee iLabs](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEgngXbpyktYw5yyXiz4ju7w_2E2IE7-_TOcfxIjGLjosHOfIAp01nAZwXr4nqk39AoBh6zJRL_AdrlgSYMbI58jvIJQZc-ZT10jWhgL1D0WQzCQ_Uzx9Ueb7C1jEbN1ONijjZ12NjUDjTkRiAF_wAD1hN3rpSiLAznpX4tLRQ_ZJfk7YAD5neZebW2_NanF/s800/Screenshot%202026-06-10%20160620.webp)

Di antara deretan toko yang jual API AI, ada satu yang namanya **iLabs**. Saya iseng klik, baca deskripsi, lihat-lihat produknya. Yang bikin saya melongo: mereka jual API dengan harga publik di website iLabs sendiri. Dan yang paling penting...

**Bayar pake QRIS.**

Wadaw. Ini yang saya mau.

![Bayar API AI Pake Qris](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEimxX39BjPiltEOBdF6ps2arTbsCwBhsHZFthalUquHKXP7rM02k2z8oMmCodsnB_oPo39d2USUsWZu-fI6epI0XsF2ICwUYy5bIG7WAll1-_bVoUd6maDNX-1bSE2C6UFlMEq6LcoWYxMZ-OZlMDqLYtzGkVjgnmH80_e-atssxev2xB1omsjThcH3d7xB/s800/Screenshot%202026-06-10%20160838.webp)

Saya langsung buka website iLabs, daftar akun, baca-baca bentar soal layanan mereka. Mereka jual berbagai macam API AI — termasuk **DeepSeek V4 Flash**, model yang sebenarnya jadi andalan OpenCode di versi terbaru. Harganya? Ramah kantong. Dan bisa pake QRIS.

Gas pol. Saya daftar, top up, bayar pake QRIS. Transaksinya cepet, langsung masuk. Nggak sampe 5 menit.

### Tes di OpenCode

Setelah saldo masuk, saya konfigurasi OpenCode pake API Key dari iLabs. Masukin endpoint, masukin API Key, ganti model ke DeepSeek V4 Flash.

Coba tes: "Generate komponen Astro untuk menampilkan daftar artikel dengan filter kategori."

*Proses...*

Muncul kodenya. Lengkap. Cepet. Nggak ada error. Nggak ada limit.

Cihuy.

Bisa beraktivitas dan berkreasi lagi.

Rasanya tuh kayak lagi *ngoding* di weekend pagi dengan segelas kopi tubruk — adem, lancar, nggak ada yang ngehalangin.

### Refleksi: Yang Saya Pelajari

Dari pengalaman ini, saya belajar beberapa hal:

1. **Limit itu bukan akhir.** Ada banyak jalan untuk tetap berkarya. Jangan nyerah pas kena limit — anggap aja alarm buat cari jalan ninja.

2. **Kadang solusi ada di tempat yang nggak disangka.** Siapa sangka penjual di Shopee bisa ngarahin saya ke penyedia API yang cocok? Dari Shopee ke website resmi, dari website resmi ke QRIS, dari QRIS ke DeepSeek yang kenceng.

3. **QRIS adalah penyelamat bapak-bapak.** Buat kita yang belum punya credit card atau males ribet urusan bank, QRIS jadi jalan keluar. Tinggal scan, bayar, selesai.

4. **Pilih model sesuai gaya belajar.** DeepSeek V4 Flash ini model yang cocok buat *vibe coding* — responsif, efisien, nggak boros token. Cocok buat yang suka trial and error kayak saya.

5. **Jangan takut cobain.** Saya sempet ragu untuk top up — takut zonk, takut rugi. Tapi dengan top up minimal via QRIS, resikonya kecil banget. Kalaupun nggak cocok, nggak perlu nangis.

### Apa Selanjutnya?

Sekarang saya udah punya akses penuh ke OpenCode dengan DeepSeek V4 Flash via iLabs. Rencana ke depan: lanjutin proyek-proyek yang sempet mandek. Ada aplikasi invoice, ada static site yang perlu di-*polish*, ada banyak eksperimen yang pengen saya coba.

Intinya: **Carpe Diem**. Begitu ada jalan, hajar aja dulu. Jangan nunggu sempurna, jangan nunggu semua alat tersedia. Kalo ada kesempatan, ambil.

Buat kalian yang juga pake OpenCode free tier dan mulai merasa terbatas, mungkin opsi ini bisa jadi pertimbangan. Atau kalo punya jalan ninja lain, share juga di komentar. Saya selalu seneng denger cerita perjuangan sesama bapak-bapak digital.

Semoga bermanfaat. Sampai jumpa di artikel berikutnya — semoga ada cerita baru tentang eksperimen gila selanjutnya.
